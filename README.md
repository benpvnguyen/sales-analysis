# Superstore Sales Analysis

An end-to-end data analysis project using Python and MySQL to uncover revenue trends and profitability insights from 10,000+ retail transactions.

## Tools & Technologies
- **Python** (pandas, matplotlib, mysql-connector)
- **MySQL** for data storage and querying
- **Jupyter Notebook** for analysis and visualization

## Key Questions Answered
- Which product category drives the most revenue?
- Which region is the most profitable?
- Which sub-categories have the highest profit margins?
- Which states are losing the most money?
- How do sales fluctuate throughout the year?

## Key Findings
- Technology is the highest revenue category
- The West region leads in profitability
- Copiers and Phones are the most profitable sub-categories
- Texas, Ohio, and Pennsylvania are the worst performing states
- Sales peak in November and December, indicating strong seasonal demand

## How to Run
1. Clone the repository
2. Install dependencies: `pip install pandas matplotlib mysql-connector-python`
3. Set up a MySQL database called `sales_analysis`
4. Run the Jupyter Notebook: `jupyter notebook sales_analysis.ipynb`
